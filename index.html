Projets

Développement d’un site de recherche de formateurs

J’ai conçu un site web en PHP pour une entreprise spécialisée dans la mise en relation avec des formateurs en informatique. Le projet inclut la gestion d’une base de données et l’application de l’architecture MVC pour une meilleure organisation du code.

Création d’une application de recherche de joueurs de football

J’ai développé une application desktop en JavaScript avec ElectronJS et MongoDB. Elle permet de rechercher des joueurs grâce à un algorithme basé sur la distance de Levenshtein, améliorant ainsi la précision des résultats.

Développement d’un dispositif BadUSB
